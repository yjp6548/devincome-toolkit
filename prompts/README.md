### 💰 2. 产品落地页文案 Prompt

```
角色：SaaS 产品文案高手，擅长用简短的文字传达价值
任务：为一个{产品类型}写落地页文案

产品信息：
- 名称：{产品名}
- 解决什么问题：{问题描述}
- 目标用户：{用户画像}
- 核心功能：{功能列表}
- 定价：{价格}

要求：
1. 一句话 Hook（3秒内抓住注意力）
2. 3个核心卖点（每个不超过一行）
3. 使用场景描述（让用户看到自己）
4. 社会证明元素（星级/评价/用户数）
5. CTA 按钮（提供3个选项）
6. FAQ（5个常见反对意见及回答）
```
